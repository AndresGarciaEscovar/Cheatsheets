\section{\quoted{REACT} Cheatsheet}

%///////////////////////////////////////////////////////////////////////////////
% Installation
%///////////////////////////////////////////////////////////////////////////////
\subsection{Installation}

To install REACT in Linux we can use the command line:
\begin{enumerate}
    \item Install the \mintinline{bash}|npm| package manager:
    \begin{mintedbox}{bash}
sudo apt install npm
    \end{mintedbox}
    %
    \item Install the \mintinline{bash}|create-react-app| package:
    \begin{mintedbox}{bash}
sudo npm install -g create-react-app
    \end{mintedbox}
    %
    \item It is recommended to install the \mintinline{bash}|node.js| package:
    \begin{mintedbox}{bash}
sudo apt install nodejs
    \end{mintedbox}
    %
    \item Check the installation:
    \begin{mintedbox}{bash}
create-react-app --version
    \end{mintedbox}
\end{enumerate}

%///////////////////////////////////////////////////////////////////////////////
% Creating an Application
%///////////////////////////////////////////////////////////////////////////////
\subsection{Creating an Application}

To create and run an application do:
\begin{enumerate}
    \item Navigate to the directory where you want to create your application:
    \begin{mintedbox}{bash}
cd <path_to_directory>
    \end{mintedbox}
    %
    \item Create the application by using the
    \mintinline{bash}|create-react-app| command:
    \begin{mintedbox}{bash}
npx create-react-app <app_name>
    \end{mintedbox}
    the \mintinline{bash}|npx| command a package runner tool that comes with
    \mintinline{bash}|npm| $5.2+$ and higher.
\end{enumerate}

%///////////////////////////////////////////////////////////////////////////////
% Running an Application
%///////////////////////////////////////////////////////////////////////////////
\subsection{Running an Application}

Once the application has been created:
\begin{enumerate}
    \item Navigate to the application directory:
    \begin{mintedbox}{bash}
cd <app_name>
    \end{mintedbox}
    %
    \item Run the application:
    \begin{mintedbox}{bash}
npm start
    \end{mintedbox}
    %
    \item Open a web browser and navigate to:
    \begin{mintedbox}{bash}
http://localhost:3000/
    \end{mintedbox}
\end{enumerate}
